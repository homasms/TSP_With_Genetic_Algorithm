# Traveling Salesman Problem using Genetic Algorithm

This repository contains a Python implementation of the **Traveling Salesman Problem (TSP)** solved using a **Genetic Algorithm (GA)**.

## Overview

The goal of the project is to find an optimized route that visits all cities exactly once and returns to the starting point, minimizing the total travel distance.

A genetic algorithm is used to iteratively improve solutions through:

* Selection
* Crossover
* Mutation
* Fitness evaluation (total path distance)

## Results

The algorithm starts with a random population of routes and gradually improves them over generations.
